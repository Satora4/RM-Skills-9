Spielregeln



Tipp Spiel

Für jeden Tipp den man abgibt, kann man 0 bis 3 Punkte erhalten.

Die Punkte werden folgendermassen verteilt:

  • Falls man das exakte Resultat (Tipp: 2:0, Resultat: 2:0) tippt, erhält man 3 Punkte.
  • Falls man die richtige Tendenz und die richtige Tordifferenz (Tipp: 2:0, Resultat 3:1; Unterschied = 2) tippt,
    erhält man zwei Punkte. Ausnahme: Endet die Partie mit unentschieden, erhält man nur einen Punkt.
  • Falls man nur die richtige Tendenz (Tipp: 2:0, Resultat: 3:2) tippt, erhält man einen Punkt.
  • Für einen falschen Tipp (Tipp: 1:0, Resultat: 0:0) gibt es keine Punkte.

Getippt werden kann bis zum Anpfiff des Spiels. Bereits getippte Spiele können bis zum Beginn der Partie geändert werden. Das heisst, dass beim Startspiel Katar gegen Ecuador (Anpfiff 20.11. um 17:00 Uhr) bis am 20.11. um 16:59 Uhr getippt werden kann.

Falls ein Spiel in die Verlängerung oder ins Elfmeterschiessen gehen sollte, zählt das Endresultat und nicht der Spielstand nach 90 Minuten.
