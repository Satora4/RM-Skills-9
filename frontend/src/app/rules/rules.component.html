Spiel Regeln


Tipp Spiel

Für jeden Tipp den man abgibt, kann man 0 bis 3 Punkte erhalten.

Sie werden folgendermaßen verteilt:

  • Falls man das exakte Resultat (Tipp: 2:0, Resultat: 2:0) tippt, erhält man 3 Punkte.
  • Falls man die richtige Tendenz und die richtige Tordifferenz (Tipp: 2:0, Resultat 3:1, Unterschied = 2) tippt,
    so erhält man zwei Punkte. Dabei gibt es eine Ausnahme: bei Unentschieden erhält man nur einen Punkt.
  • Falls man nur die richtige Tendenz (Tipp: 2:0, Resultat: 3:2) tippt, erhält man einen Punkt.
  • Für einen falschen Tipp (Tipp: 1:0, Resultat: 0:0) gibt es keine Punkte.

Getippt werden kann, bis zu Spielbeginn. Bereits getippte Spiele können ebenfalls bis Spielbeginn geändert werden. Das heißt, dass das Startspiel Senegal gegen Niederlande (21.11, 11:00) bis 21.11, 10:59 getippt werden könnte.

Falls ein Spiel in die Verlängerung oder ins Elfmeterschießen gehen sollte, zählt das Endresultat und nicht der Spielstand nach 90 Minuten.
